###OBJECTIVE###

Develop an interactive business intelligence dashboard to visualize country-wise COVID-19 statistics, enabling users to explore global trends, compare regions, and derive actionable public health insights.

###DATASET###

Country-wise COVID-19 dataset containing global statistics such as confirmed cases, deaths, recoveries, active cases, new cases, and WHO regional classifications.

###ANALYSIS PERFORMED###

Data cleaning & preparation

Removed inconsistencies and verified numerical fields

Converted columns to correct data types

Checked for missing or invalid values

Dashboard design

Created KPI cards to show global totals

Built bar, pie, line, and table visuals

Applied region-wise filtering using slicers

Interactivity implementation

Enabled cross-filtering between visuals

Added WHO Region slicer for dynamic filtering

Structured a single-page layout for fast insight access

###KEY INSIGHTS###

The Americas and Europe regions account for the highest number of confirmed cases.

Some countries have high death rates despite moderate case counts, indicating healthcare challenges.

Regions with high active cases signal ongoing outbreak risks.

Recovery percentages vary significantly, reflecting differences in healthcare infrastructure.
