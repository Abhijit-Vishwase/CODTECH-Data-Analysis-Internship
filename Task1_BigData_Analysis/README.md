# Task-1: Big Data Analysis Using PySpark

## Internship Project – CODTECH

### Objective
Perform scalable analysis on a large dataset using PySpark.

### Dataset
E-commerce transaction dataset containing 119,614 records.

### Analysis Performed
- Total revenue calculation
- Revenue by state
- Payment method distribution
- Top product categories

### Key Insights
- Total revenue ≈ 17.36 million
- São Paulo generated highest sales
- Credit card is most used payment method
- Toys is the most purchased category

### Tools Used
- Python
- PySpark
- Jupyter Notebook
